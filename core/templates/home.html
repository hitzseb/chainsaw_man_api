Chainsaw Man API

La API de Chainsaw Man proporciona acceso a información detallada sobre el manga "Chainsaw Man". Esta API permite realizar solicitudes GET desde cualquier servidor sin necesidad de credenciales, facilitando el acceso a datos relacionados con el manga, personajes, y más.

Endpoints disponibles:

{{server_url}}api/saga : listado completo de las sagas.

{{server_url}}api/arc : listado completo de los arcos.

{{server_url}}api/volume : listado completo de los volúmenes.

{{server_url}}api/manga : listado completo de los manga. Sólo su número, título y portada.

{{server_url}}api/manga/detail/number : detalles del manga con el atributo 'number'.

{{server_url}}api/character : listado completo de los personajes. Sólo su nombre y retrato.

{{server_url}}api/character/detail/name : detalles del personaje con el atributo 'name'.
